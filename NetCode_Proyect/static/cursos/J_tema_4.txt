<h3>tema 4: Los operadores Matematicos</h3>
<br></br>
Java provee un amplio conjunto de operadores para utilizarse en la manipulación de variables.
 Un valor utilizado en cualquier lado del operador es llamado un operando.
<br></br>
Por ejemplo, en la expresión siguiente, los números 6 y 3 son operandos del operador suma:
<br></br>
int x= 6 + 3;
<br></br>
Operadores aritméticos Java:<br></br><br></br>
+ adición<br></br>
- sustracción<br></br>
* multiplicación<br></br>
/ división<br></br>
% módulo<br></br>
<br></br>
Adision<br></br>
El operador + suma dos valores, como dos constantes, una constante y una variable, o una variable y otra variable
<br></br>
Sustraccion<br></br>
El operador - sustrae un valor de otro
<br></br>
Multiplicacion<br></br>
El operador * Multiplica dos valores.
<br></br>
Division<br></br>
El operados / divide un valor entre otro
<br></br>
Modulo<br></br>
El operador matemático módulo (o resto) ejecuta una división entera de un valor entre otro valor, y retorna el resto
de esa división.<br></br>
El operador para la operación de módulo es el carácter de porcentaje (%).
