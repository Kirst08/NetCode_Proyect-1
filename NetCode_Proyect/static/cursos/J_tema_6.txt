 strings

Un String (cadena de texto) es un objeto que representa una secuencia de caracteres.
Por ejemplo, "Hola" es un string de 4 caracteres.

Por ejemplo:

string s = "NetCode"

se puede definir un string vacio EJemplo: string str = "";
