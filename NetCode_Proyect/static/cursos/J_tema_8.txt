Declaraciones if anidadas

Puedes utilizar una declaracion if-else dentro de otra declaracion if o else.

Por ejemplo:

int age = 25;
        
if(age > 0) {
            
    if(age > 16) {
                
        System.out.println("Welcome!");
            
     } else {
               
          System.out.println("Too Young");

     }
        
} else {

     System.out.println("Error");
}
