Introduccion a Java

Java en un lenguaje moderno de programación de alto nivel, diseñado al inicio de los años 90 por Sun Microsystems y actualmente propiedad de Oracle. Java es una Independiente de la Plataforma, lo que significa que ¡sólo necesitas escribir el programa una vez para poderlo ejecutar en varias plataformas diferentes! Java es portable, robusto y dinámico, con la habilidad de ajustarse a las necesidades de virtualmente cualquier tipo de aplicación Java es utilizado para desarrollar aplicaciones para Android de Google, diversas aplicaciones para Escritorio, como reproductores multimedia, programas antivirus, Aplicaciones Web, Aplicaciones Empresariales (por ejemplo, Bancarias) ¡y muchas más!

A Hello World Program

Tu primer programa Vamos a comenzar creando un programa sencillo que imprima "Hello World" en la pantalla. class MyClass { public static void main(String[ ] args) { System.out.println("Hello World"); } } En Java, cada línea de código que se puede ejecutar necesita estar dentro de una clase. En nuestro ejemplo, nombramos la clase MyClass. Aprenderás más sobre clases en los próximos módulos. En Java, cada aplicación tiene un punto de entrada, o un punto de partida, el cual es un método llamado main.
